# Purpose
Create a design for the Ollama server feature in roborev project.

# Process
1. Review the [pull request](https://github.com/roborev-dev/roborev/pull/149). 
- Try to back out from this the requirements of the feature.
- See if the development points to any difficulties to avoid.
2. Review the current code in the project @https://github.com/roborev-dev/roborev.git
3. Using an Ollama server should be similar to using any other LLM server
- See how much is similar in interface
- Identify the differences between an ollama-based model and one from a commercial LLM.
